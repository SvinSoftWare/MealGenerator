# MealGenerator
Creation of MealApp

Shows a list of desserts and their recipes/ingredients for people to learn how to cook

Instructions:
  1. Download Zip File
  2. Open MealGenerator.xcodeproj
  3. Run App From XCode
  4. IPhone 14 Pro IOS
